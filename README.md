## Описание проекта
Проект представляет из себя сервис онлайн библиотеки со следующими функциями:
- Авторизация/Регистрация
- Сохранение/Удаление/Изменение/Прочтение книг
- Возможность семантического и полнотекстого поиска
- Возможность сохранения истории поиска
- Возможность фильтрации поиска
- Создание заметок и закладок в книгах
- Получение ачивок
- Сохранение прогресса чтения
- Добавление книг в список любимых

## Локальный запуск
Для локального запуска скачайте любым удобным вам способом код из ветки main, а затем, находясь в корне проекта пропишите
```
docker compose up --build
```
После этого у вас запуститься приложение на всех ваших ip-адресах на порту 3000 <br><br>
Пример
```
http://127.0.0.1:3000
```

## Используемые хранилища
В нашем проекте мы используем:
- Minio для хранения самих файлов и обложек
- PostgreSQL для пользователей и метаданных о книгах
- Какое-то хранилище эластика

## Схема таблиц бд
### users
- id (PK)
- username
- email
- password

### books
- id (PK)
- title
- author
- language
- genre
- minio_key
- cover_key

### favourites
- id (PK)
- user_id (FK)
- book_id (FK)
